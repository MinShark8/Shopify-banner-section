Installation Steps

Step 1: Upload Assets

1. Go to Shopify Admin → Online Store → Themes
2. Click ⋯ → Edit code
3. Open the Assets folder
4. Right-click on the Assets folder and select “Upload...” to add the files to your theme.
Files:
    + el-banner.css
    + el-banner.js

Step 2: Upload  Section
1. In Edit code, open the Sections folder
2. Right-click on the Sections folder and select “Upload...” to add the files to your theme.
Files:
+ el-banner.liquid
